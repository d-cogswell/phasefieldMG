# README #


### What is this repository for? ###

This repository hosts code for solving phase-field models (and other PDE's) using multigrid methods.

### How do I get set up? ###

Requirements:

* C/C++ compiler
* Magick++ ([http://www.imagemagick.org/Magick++/](http://www.imagemagick.org/Magick++/))
